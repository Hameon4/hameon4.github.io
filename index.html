Projects

SCOOTER INCIDENT REPORTING SYSTEM

This was my Senior Project where my team and I designed an attachable device to an e-scooter, including the algorithm whose duty is to detect a fall, collision, or a crash from the e-scooter and transmit the data to the IoT platform via Wi-Fi. The IoT platform utilized was Thinger.io.

The algorithm we developed tracks the movement of the scooter. High spikes on the IoT platform's dashboard indicate a high possibility of an accident. The algorithm then sends an email providing an alert, including the device's ID and location so help can arrive ASAP.

Much thought came among my team and me on improving the product, such as adding a panic button that can reduce the chances of false negatives. We also thought of incorporating a speaker/mic setup using VoIP to enable continuous communication with the monitoring team when required. And when possible, utilize AI to teach the system how falls happen and make it predict them when they occur.

On-Board Computer (OBC) Simulator

As my main task during my internship at the National Space Science and Technology Center (NSSTC) at UAEU, I developed a desktop (GUI) simulating software, using Python and Arduino, that acts as a replica of the telecommand/telemetry process in the satellite with UART as the communication protocol.

The initial prototype was a CLI app, which I then modified to a second, and the go-to prototype: the GUI app. The framework used was Python's Tkinter.

HOTEL RESERVATION WEBSITE

Built a full-stacked website using frontend tools (HTML, CSS, JS) with PHP as the backend and MySQL as the database. This was my end-of-term Software Engineering Fundamentals course project in which the goal was to provide the software design planning (i.e. purpose/scope of the system, functional/non-functional requirements, use-case/class/sequence diagrams).

Following that was demonstrating it by building an app (either web-app or GUI), which in my case was a website. I managed to display the data from the database into the website UI with only the admin's account can access. I also managed to automate the user creation process and save the person's information into the database.
